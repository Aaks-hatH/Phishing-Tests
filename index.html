CoreGuard Extended Test Lab

Open each test in a new tab. The extension should detect and block/alert for the scenarios below.

Typosquat / Brand Impersonation

Page displays a brand logo and password field on a non-official domain (should be blocked).

Open Typosquat Login Triggers: brand visuals + password field, SLD impersonation

Credential Exfil Form (supabase-like)

Form that appears to POST credentials to a supabase-like endpoint (should be blocked).

Open Exfil Form Triggers: external DB indicators + credential fields

Shortener / Tunner redirect (meta refresh)

Redirects via meta-refresh to a tunneler/shortener host (escalate / block).

Open Shortener Redirect Triggers: meta-refresh -> tunneler (trycloudflare/loclx)

Drive-by Download (programmatic)

Auto-creates a Blob and triggers a download (should be canceled by downloads protection).

Open Download Test Triggers: programmatic download, suspicious filename

Popup Chain Attack

Opener spawns a child popup that navigates to a malicious/impersonation host. Extension should neutralize source.

Open Popup Chain Triggers: popup spawn -> suspicious host

Iframe Phishing

Embed an iframe that contains a credential form. Extension should inspect and block.

Open Iframe Phish Triggers: embedded credential harvesting

Cloaked Anchor

Visible link text says "paypal.com" but href points elsewhere (anchor cloaking detection).

Open Cloaked Anchor Triggers: anchor text vs href mismatch

Obfuscated JS Redirect

Performs timed, obfuscated JS redirect. Extension should catch redirect behavior.

Open Obfuscated Redirect Triggers: suspicious JS redirect behavior
